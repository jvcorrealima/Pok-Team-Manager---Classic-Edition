# 🎮 PokéTeam Manager

Uma interface nostálgica e funcional inspirada na era clássica de Pokémon para Game Boy. Este projeto foi desenvolvido para permitir que treinadores organizem seus times de 6 Pokémon, detalhando níveis, tipos elementais e conjuntos de habilidades.

## 🚀 Demonstração
O projeto utiliza a estética clássica da Pokédex, com foco em uma experiência visual imersiva para fãs da franquia.

## 🛠️ Tecnologias Utilizadas
*   **HTML5:** Estrutura semântica dos dados do treinador e dos Pokémon.
*   **CSS3:** Layout avançado com **CSS Grid** e **Flexbox**, além de tipografia customizada com a fonte *'Press Start 2P'*.
*   **JavaScript:** Utilizado para otimizar a repetição dos blocos de código e manipulação dinâmica dos cards.

## 🎨 Destaques do Projeto
*   **Design Retrô:** Interface baseada no layout original do Game Boy.
*   **Gestão de Time:** Espaço detalhado para 6 Pokémon com 4 slots de ataques cada.
*   **PC Storage:** Seção dedicada para "Box Reserva", simulando o sistema de armazenamento dos jogos.
*   **Responsividade:** Organização em colunas que se adaptam para manter a leitura clara.
